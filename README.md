HOW TO RUN
----------
1. Save the code as an `.html` file (e.g., `secret-santa.html`).
2. Double-click the file to open it in any modern web browser (Chrome, Firefox, Safari, Edge).
3. No internet connection is required for the logic, BUT an internet connection IS required to load the external libraries (Confetti and SheetJS) via CDN.

HOW TO USE
----------
1. **Add Participants:**
   - Type a name in the text box and press Enter (or click '+').
   - OR upload an Excel file (see format below).
2. **Review the Pool:**
   - Added names appear as "chips" in the center area.
   - You can remove a name by clicking the 'X' next to it.
3. **Generate Pairs:**
   - Once you have 2 or more names, the "Reveal Next Pair" button becomes active.
